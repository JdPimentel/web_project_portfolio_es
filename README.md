# Portafolio Juan Pimentel

## Descripción

Sitio web de portafolio personal creado como proyecto del Sprint 4 del bootcamp TripleTen.

## Características del proyecto

- Diseño adaptado para escritorio, tablet y móvil.
- Uso de gradientes radiales para crear efectos de luz en el fondo.
- Links con subrayado personalizado en degradado.
- Efectos hover con transiciones suaves.
- Tarjetas de proyectos con imágenes, tecnologías usadas y enlaces externos.
- Fuentes locales cargadas desde la carpeta del proyecto.

## Tecnologías utilizadas

- HTML
- CSS
- Flexbox
- Media queries
- Diseño responsivo
- Gradientes con CSS
- Efectos hover
- Transiciones
- Pseudo-elementos
- Fuentes locales con @font-face
- Figma
- Git y GitHub
